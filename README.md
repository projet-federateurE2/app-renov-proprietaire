# Noms des branches
- feat/...   -> une nouvelle fonctionnalité ajoutée au projet
- fix/...    -> un fix de bug
- style/...  -> changement des styles sans modifier les fonctionnalités
- refactor/... -> réorganisation du code
- doc/... -> ajout de nouvelles documentations

- En anglais !
